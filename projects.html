Projects

Case Analysis Project

In this project we looked at the company SmoothPay. SmoothPay is a mobile-payment company that is based in Canada. We looked at SmoothPay to analyze their current business model, strategies, and processes. We then offered suggestions on how to best improve the company and help it grow into the future.

Web Development Project

For this project we developed a website for a famous Pittsburgh business: Prantl's Bakery. This project taught us how to design, develop, and code a front-end website. It taught a lot about the whole process and was educational.

Database Project

For this project, we were tasked with modeling important business data and to create a database that performs this function.
